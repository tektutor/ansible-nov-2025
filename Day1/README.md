## Python Overview
<pre>
- is a high level scripting programming language
- dynamically typed language
- data type
  - int
  - float
  - string
  - bool
  - sequential types
    - list
    - tuple
    - range
  - maps
    - dict
    - set
  - binary types
    - bytes
    - bytearray
- it is used for developing
  - console/command-line applications
  - AI/ML 
  - Web applications
  - Games
  - Academic
  - pattern matching
  - embedded projects
- python is supported in following OS
  - Unix
  - Linux
  - Mac
  - Windows
</pre>
